## 🧩 What is CuriosPaper?

CuriosPaper adds a complete **accessory slot system** to your Minecraft server. Players can equip rings, necklaces, capes, belts, charms, and more through a clean, intuitive GUI — no mods required.

Think of it as **Baubles / Curios** — but for Paper & Spigot servers, fully server-side, and built with a powerful API that other plugins can hook into.

## ⚡ Features at a Glance

| | Feature | |
|:---:|---|---|
| 🎒 | **9 Accessory Slot Types** | Head, Necklace, Back, Body, Belt, Hands, Bracelet, Ring, Charm |
| ✨ | **Ability System** | Attach potion effects & attribute modifiers to any accessory |
| 🛠️ | **In-Game Item Editor** | Create & configure accessories visually — no YAML editing |
| 📦 | **Recipe System** | Shaped, shapeless, furnace, blast furnace, smoker, anvil & smithing |
| 👹 | **Mob Drops** | Custom accessories drop from any mob with configurable chances |
| 🏘️ | **Villager Trades** | Add accessories to villager trade pools |
| 🎨 | **Resource Pack Hosting** | Built-in HTTP server auto-generates & serves the resource pack |
| 🪂 | **Elytra Back Slot** | Equip elytra in the back accessory slot (1.21.3+) |
| 🔌 | **Developer API** | Full API for other plugins to manage accessories & hook events |
| 📊 | **Rearrangeable GUI** | Admins can rearrange the accessory menu layout in-game |

## 🎮 How It Works

1. Players type **`/baubles`** to open the accessory menu
2. Click a slot icon (e.g. Ring 💍) to open that slot's inventory
3. Place valid accessories into the slot — abilities activate instantly
4. Close the menu and enjoy the effects!

## 📥 Quick Start

1. Download **CuriosPaper.jar**
2. Drop it into your server's `plugins/` folder
3. Restart the server
4. Type `/baubles` in-game — done!
