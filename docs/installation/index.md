## Quick Install

1. Download `CuriosPaper.jar` from the [releases page](https://github.com/Brothergaming52/CuriosPaper/releases)
2. Place it in your server's `plugins/` folder
3. Restart the server
4. (Optional) Configure the resource pack in `config.yml`

That's it! CuriosPaper will generate default configuration files and be ready to use.
